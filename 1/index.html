Java

Lenguaje de programación orientado a objetos, ampliamente utilizado en aplicaciones empresariales, móviles y sistemas embebidos. Destaca por su portabilidad y robustez.

Python

Lenguaje interpretado, versátil y fácil de aprender. Muy popular en ciencia de datos, inteligencia artificial, desarrollo web y automatización.

C++

Lenguaje de programación de alto rendimiento, ideal para desarrollo de sistemas, videojuegos y aplicaciones que requieren eficiencia y control de recursos.

JavaScript

Lenguaje esencial para el desarrollo web frontend, permite crear interfaces interactivas y dinámicas en navegadores. También se usa en el backend con Node.js.

Rust

Lenguaje moderno enfocado en la seguridad y el rendimiento. Muy valorado para sistemas críticos y aplicaciones donde la concurrencia y la ausencia de errores de memoria son clave.
